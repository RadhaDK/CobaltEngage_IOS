How to run the iOS sample application

1/ Open BasicSample.xcodeproj in Xcode
2/ Connect your iOS device to your computer
3/ Enter your Gimbal API key for the sample app in AppDelegate.m
4/ Set the run target of the app to the connected phone
5/ Run it
